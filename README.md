# ✅ To-Do-App

A clean, intuitive task management applicaion that lets you organize your daily tasks easily. This project demonstrates
core JavaScript functionally including DOM manipulation and localStorage for data persistence.

## 🚀 Live Demo

[View Live](https://mahdihassani1408-rgb.github.io/To-Do-App)

## ✨ Features

- Add Tasks - Quickly add new tasks to your list with a single input field
- Delete Tasks - Remove individual tasks when they're no longer needed
- Mark as Completed - Toggle task completion status with a click, visually striking through completed tasks
- LocalStorage Persistence - Your tasks are automatically saved in your browser, so they remain even after refreshing the page
- Filter/Sort Options(if applicable) - View all, active, or completed tasks

## 🛠 Tech Stack

- HTML5 - Semantic structure for accessibility and SEO
- CSS3 - Custom styling with Flexbox for modern, responsive layouts
- JavaScript(ES6)
    - DOM Manipulation for dynamic task rendering
    - localStorage API for data persistence
    - Event delegation for efficient event handling

## 🏁 Getting Started

Follow these simple steps to run the project locally

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, or Safari)
- A code editor (recommended: VS Code)

### Installation

1. Clone the repository

``` bash
git clone https://github.com/mahdihassani1408-rgb/To-Do-App.git
cd To-Do-App
```

2. Open the application
    - Simply open index.html in your browser, or use a local development server (like the Live Server extension in VS code) for the best experience

3. Start managing tasks
    - Type a task in the input field and click "Add"
    - Click on a task to toggle its completion status
    - Click the delete button

Building this To-Do-App helped me understant several fundamental web development concepts:
